UNIX COMMANDS NOTES
-------------------

1. ls
   Lists files and directories in the current directory.
   Example:
   ls
   ls -l
   ls -a

2. pwd
   Displays the present working directory.
   Example:
   pwd

3. cd
   Used to change the directory.
   Example:
   cd Documents
   cd ..
   cd ~

4. mkdir
   Creates a new directory.
   Example:
   mkdir test
   mkdir dir1 dir2

5. rmdir
   Removes an empty directory.
   Example:
   rmdir test

6. touch
   Creates an empty file.
   Example:
   touch file.txt

7. cat
   Displays file content and is also used to write into a file.
   Example:
   cat file.txt
   cat > file.txt
   cat >> file.txt

8. echo
   Prints text on terminal and writes text to a file.
   Example:
   echo "Hello Unix" > file.txt
   echo "New line" >> file.txt

9. cp
   Copies files or directories.
   Example:
   cp file1.txt file2.txt

10. mv
    Moves or renames files.
    Example:
    mv old.txt new.txt

11. rm
    Deletes files or directories.
    Example:
    rm file.txt
    rm -r foldername

12. chmod
    Changes file permissions.
    Example:
    chmod 755 file.sh

13. grep
    Searches for a pattern in a file.
    Example:
    grep "error" log.txt

14. head
    Displays first 10 lines of a file.
    Example:
    head file.txt

15. tail
    Displays last 10 lines of a file.
    Example:
    tail file.txt

16. wc
    Counts lines, words, and characters.
    Example:
    wc file.txt

17. clear
    Clears the terminal screen.
    Example:
    clear

18. man
    Displays the manual of a command.
    Example:
    man ls

-------------------
END OF NOTES
